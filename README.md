P(X=r) = nCr · pʳ · (1-p)ⁿ⁻ʳ

P = P(X=2) + P(X=3) ... up to X
so X = bid up to number of dice, to get all success conditions

r = successes (bid)
nCr = n choose r
n = number of dice
p = percentage of success => 1/6 for six sided dice

nCr = n! / [(n - r)! * r!]
